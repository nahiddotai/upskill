# Welcome to Upskill

If you can read this skill in a client, your sync pipeline works: this file
lives in your personal `upskill-skills` repo and was delivered here by the
Upskill plugin.

Try the loop:

1. Edit this file in your workspace (`~/.upskill/workspace/plugins/skills/skills/welcome/SKILL.md`).
2. Say **"upskill sync"** in any client.
3. Open another client and run `/upskill:pull` — your edit is there.

When you're done with it, delete this folder and sync again.
